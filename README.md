# 날씨 어플 클론 코딩

vue.js를 사용하여 아이폰 기본 날씨 어플 프론트 클론 코딩 프로젝트

- 기간 : 2022-06-12 ~ 2022-06-19(1차)
- 목표 : 날씨 어플 클론코딩을 통한 vue.js 활용과 날씨 openAPI 활용
- 사용기술 : vue3

## 날씨 어플 기능
- 사용자 지역을 보여줍니다.
- 현재 지역명과 온도, 날씨를 안내해줍니다.
- 시간별로 앞으로의 온도,날씨를 예상해줍니다.

### 미정 기능
- 대기질 확인
- 기온 지도 확인
- 자외선 지수 확인
- 일몰, 일출 시간 확인
- 바람 확인
- 강우 확인
- 체감 온도 확인
- 습도 확인
- 가시거리 확인
- 기압 확인
